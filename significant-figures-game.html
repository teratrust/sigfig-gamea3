解説

有効数字とは

① 有効数字とは何か？

測定値を表すとき、意味のある（信頼できる）桁数のことを有効数字といいます。測定には必ず誤差が伴うため、「どこまで正確に測れたか」を示す必要があります。

📏 定規で 2.35 cm と読めたとき
「2.3」→ 目盛りで確実に読める
「5」→ 目盛りの間を目測した推定値
→ 有効数字 3桁
② 有効数字の桁数の数え方（基本ルール）
  • 10以外の数字はすべて有効数字
    例：3.14 → 3桁、52.7 → 3桁
  • 2数字と数字の間にある0は有効数字
    例：1.05 → 3桁、1006 → 4桁
  • 3小数点以下の末尾の0は有効数字
    例：1.500 → 4桁、0.720 → 3桁
  • 4整数の末尾の0は有効数字かどうか曖昧
    例：1500 → 2桁〜4桁（不明）
  • 5数の先頭の0は有効数字ではない
    例：0.0025 → 2桁、0.030 → 2桁
③ 科学的記数法（指数表示）

非常に大きい数・小さい数は a × 10n（1 ≦ a ＜ 10）の形で表します。有効数字の桁数が一目でわかります。

0.000456 = 4.56 × 10⁻⁴（3桁）
123000 = 1.23 × 10⁵（3桁と明示する場合）
6.022 × 10²³ → 4桁（aの部分が有効数字）
① 掛け算・割り算のルール

答えの有効数字の桁数は、計算した数のうち最も桁数が少ないものに合わせます。

2.5（2桁）× 3.42（3桁）= 8.55
→ 最小桁数は 2桁 → 答えは 8.6
12.0（3桁）÷ 4.0（2桁）= 3.0
→ 最小桁数は 2桁 → 答えは 3.0
6.0（2桁）÷ 3.0（2桁）= 2.0
→ 最小桁数は 2桁 → 答えは 2.0（「2」だと1桁になりNG！）
② 足し算・引き算のルール

答えの小数点以下の桁数は、計算した数のうち最も小数点以下の桁数が少ないものに合わせます。

12.52（小数2桁）+ 1.7（小数1桁）= 14.22
→ 小数の最少は 1桁 → 答えは 14.2
100.0（小数1桁）− 2.51（小数2桁）= 97.49
→ 小数の最少は 1桁 → 答えは 97.5
③ 四捨五入の注意点

桁を丸めるときは四捨五入を使います。「5」のとき切り上げる点に注意！

8.350... → 小数1桁に → 8.4（5は切り上げ）
8.349... → 小数1桁に → 8.3（4は切り捨て）

実際の計算：8.4×1... 2.4×3.75 = 9.0
答：9.0（2桁）

8.56（小数2桁）− 1.3（小数1桁）= 7.26
小数1桁に丸める → 答：7.3
